このアカウントを削除しますか？

daily-note.yml — GitHub Actionsワークフロー
name: Daily Note Generator

on:
  schedule:
    - cron: '0 20 * * *'  # 毎朝 05:00 JST
  workflow_dispatch:        # 手動実行も可能

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
      - run: npm install
      - name: Generate Daily Notes
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
          GEMINI_API_KEY: ${{ secrets.GEMINI_API_KEY }}
          NOTE_EMAIL: ${{ secrets.NOTE_EMAIL }}
          NOTE_PASSWORD: ${{ secrets.NOTE_PASSWORD }}
        run: node src/daily-auto.js
      - uses: actions/upload-artifact@v4
        with:
          name: daily-notes-${{ github.run_id }}
          path: output/drafts/
このファイルを .github/workflows/daily-note.yml として配置してください。
